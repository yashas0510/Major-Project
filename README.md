# CyberSec Deepfake Detector 🛡️

An explainable AI (XAI) deepfake detection system that analyses both images and videos, providing plain-English explanations for every prediction.

---

## What it does

- Detects deepfake faces in **images** (JPG/PNG) and **videos** (MP4/AVI)
- Shows **Grad-CAM heatmaps** highlighting which facial regions influenced the decision
- Runs **4 XAI checks** — skin tone consistency, eye blinking, lip movement, and facial boundary artifacts
- Presents results in **plain English** so anyone can understand why a face was flagged
- Full **Flask web app** with a clean minimal frontend

---

## How it works

```
Video/Image
    │
    ▼
InsightFace (face detection + crop)
    │
    ▼
XceptionNet (spatial feature extraction per frame)
    │
    ▼
LSTM (temporal sequence modelling across frames)
    │
    ▼
Binary classifier → Real / Fake
    │
    ├── Grad-CAM heatmap (which facial regions mattered)
    │
    └── XAI Analysis
            ├── Skin tone consistency (HSV frame-to-frame)
            ├── Eye blink detection (dlib EAR ratio)
            ├── Lip-sync analysis (mouth landmark motion)
            └── Boundary artifact detection (Laplacian variance)
```

## Tech stack

| Component | Library |
|---|---|
| Deep learning framework | PyTorch |
| Feature extractor | XceptionNet |
| Temporal modelling | LSTM |
| Face detection | InsightFace (buffalo_l) |

| Frontend | HTML / CSS / Vanilla JS |

## Setup

### 1. Clone the repo
```bash
git clone https://github.com/<your-username>/cybersec-deepfake-detector.git
cd cybersec-deepfake-detector
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download model checkpoints
The trained model weights are hosted separately due to file size limits.

👉 **[Download checkpoints from Google Drive](#)** 

Place them as:
```
checkpoints/
├── video/best_model_video.pt
└── image/best_model_image.pt
```

### 4. Download dlib landmark model
```bash
curl -L -o backend/shape_predictor_68_face_landmarks.dat.bz2 \
  "https://github.com/davisking/dlib-models/raw/master/shape_predictor_68_face_landmarks.dat.bz2"

python -c "import bz2, shutil; shutil.copyfileobj(bz2.open('backend/shape_predictor_68_face_landmarks.dat.bz2','rb'), open('backend/shape_predictor_68_face_landmarks.dat','wb'))"
```

### 5. Run
```bash
cd backend
python app.py
```

Open `http://127.0.0.1:5000` in your browser.

## Usage

1. Open the web UI
2. Choose **Image** or **Video** tab
3. Upload a file and click **Analyze**
4. View the prediction and Grad-CAM heatmaps
5. Click **View XAI Report** for the full plain-English explanation

---

## XAI Report

After each analysis, visit:
```
http://127.0.0.1:5000/report/<job_id>
```

The report shows:
- Overall verdict (Real / Fake) with confidence
- Plain-English summary of why the model decided
- Grad-CAM heatmap with explanation of what the colours mean
- 4 feature cards — each explaining what was checked, what was found, and why it matters
- Technical scores hidden in a collapsible section for developers

## Dataset

Trained on the **FaceForensics++** dataset.
- `train/real/` and `train/fake/` — training videos
- `val/real/` and `val/fake/` — validation videos
- `frames/` — extracted face crops (generated by InsightFace preprocessing)

---

## Known limitations

- Works best on frontal face videos with good lighting
- Lip-sync and blink checks require dlib landmarks — may fail on low-resolution or heavily compressed videos
- Audio-based lip-sync is not yet implemented (visual only)
- Not tested on GAN types beyond FaceForensics++ (e.g. diffusion-based fakes)
